# Q22 | Labels & Annotations

Add label protected: true to all Pods in Namespace sun with label type: worker or type: runner. Then annotate them with protected: 'do not delete this pod'.

This scenario is based on a real CKAD Killer Shell exam question. Use the terminal on the right to complete the task.

> **Tip:** `kubectl` is aliased to `k`. Tab completion is enabled.
